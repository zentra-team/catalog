# Zentra Catalog

Публичный **каталог модулей Zentra CMF** — список того, что доступно для установки.
Панель управления читает отсюда `catalog.json` и показывает карточки модулей
во вкладке «Модули → Доступные».

⚠️ Здесь **нет кода модулей** (он в приватном репозитории). Только витринные данные:
названия, версии, описания, цены. Поэтому репозиторий публичный — секретов нет.

## catalog.json

```json
{
  "schema_version": 1,
  "modules": [
    {
      "sys_name": "search",
      "name": "Поиск по сайту",
      "version": "1.0.0",
      "description": "Краткое описание для карточки.",
      "author": "Zentra Team",
      "is_paid": false,
      "price": 0,
      "currency": "RUB",
      "min_core_version": "1.1.0"
    }
  ]
}
```

Поля карточки:

| поле               | назначение                                            |
|--------------------|-------------------------------------------------------|
| `sys_name`         | системное имя (совпадает с папкой модуля)             |
| `name`             | название для пользователя                             |
| `version`          | текущая версия                                         |
| `description`      | описание для карточки                                  |
| `author`           | автор                                                  |
| `is_paid`          | платный (`true`) или бесплатный (`false`)             |
| `price`            | цена в рублях (для платных; для бесплатных `0`)       |
| `currency`         | валюта (`RUB`)                                         |
| `min_core_version` | минимальная версия ядра Zentra                        |

**Цена живёт здесь, а не в коде модуля** — чтобы менять её без перевыпуска модуля.

Файл наполняется автоматикой из приватного репозитория модулей при выпуске версии.
